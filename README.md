# frontend

이 저장소는 React와 Vite로 구축된 프론트엔드 프로젝트를 관리합니다.

## 기술 스택

이 프로젝트는 다음과 같은 기술을 기반으로 합니다:

-   **React**: 사용자 인터페이스 구축을 위한 JavaScript 라이브러리
-   **Vite**: 빠르고 경량화된 빌드 도구
-   **ESLint**: 코드 품질 관리를 위한 린트 도구

## 프로젝트 실행 방법

1.  의존성 패키지 설치:
    ```bash
    npm install
    ```
2.  개발 모드 실행:
    ```bash
    npm run dev
    ```
3.  프로덕션 빌드:
    ```bash
    npm run build
    ```
